<h1>Página de Usuario</h1>
  <figure>
    <p>
    <img align="left" src="https://github.com/Matii111/reactTest/blob/master/mockPaginaWeb/partes/sushiCosoUser.jpg?raw=true" width="400">

  <p align="left">
    Para la sección de usuario se decide elaborar una nueva página donde si bien se mantiene el mismo header y footer, cambia la manera de navegar la página pues las opciones en este caso están al costado izquierdo de manera vertical. Las opciones están  divididas en los grupos: Datos de usuario, que incluye nombre, apellido, y fecha de nacimiento del usuario. Datos de despacho, incluye direcciones guardadas. Datos de facturación, incluye métodos de pago del usuario. 
Por otro lado, está la sección de historial de compras el cual incluye, como el nombre lo indica, todas las compras realizadas con esta cuenta, en este lugar al escoger una compra se abre un popup con los detalles de la misma junto a un formulario para solicitar reembolso.
